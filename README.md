# Multi-Utility Python Program

This repository contains a Python program that combines three useful utilities for beginners:

## Programs Included
1. **Student Mark Manager**
   - Enter marks for multiple subjects
   - Calculates total, average, max, min
   - Counts even and odd marks

2. **Number Analyzer**
   - Enter a list of numbers
   - Finds even, odd numbers
   - Calculates sum, average, max, min

3. **Simple Calculator**
   - Performs basic arithmetic: +, -, *, /
   - Handles division by zero

## How to Run
1. Clone the repository
2. Open `all_programs.py` in Python 3
3. Run the program and choose from the menu
